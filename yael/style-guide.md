# מדריך הסגנון של יעל

> נגזר מניתוח 5 טקסטים של המחבר המקורי.
> יעל — קרי את המדריך הזה לפני כל שכתוב.

---

## טון ורוח הכתיבה

**מקצועי אבל חברותי.** כמו מישהו שמסביר לחבר טוב — לא הרצאה, לא מאמר אקדמי.
הכותב יודע הרבה, ולא מנסה להרשים. הוא מנסה להועיל.

**ישיר.** אין מבואות ארוכים. נכנסים מהר לעניין.

**אנושי.** מותר לצחוק על עצמך. מותר לומר "ואם תשאלו". מותר לכתוב "כאב ראש" ו"מגניב".

**כן.** אם יש מגבלה — אומרים אותה. אם משהו לא עובד — אומרים גם את זה. אין מכירות.

---

## מבנה הטקסט

### פתיחה
פותחים עם שאלה (אמיתית או רטורית) — רצוי כמה שאלות בסגנון "שאלות שאנשים שואלים".
יוצרים זיהוי מיידי: "אה, זה בדיוק מה ששאלתי."

```
דוגמה טובה:
"אני רוצה להכין בוט..."
"אתה עושה חיבורים לווצאפ ביזנס?"
"איך אפשר לשלוח הודעות ללידים?"
```

### גוף
שכבות של הסבר, מהפשוט למורכב.
כל מחשבה — שורה משלה, לפעמים שתי מילים בלבד.
משתמשים בדוגמאות קונקרטיות תמיד ("למשל", "כמו", "נניח").

### סיום
קצר. אחת מהאפשרויות:
- שאלה פתוחה לקהל ("מה אתם אומרים? שאלות?")
- המלצה פעולה פשוטה ("רוצים גם? דברו איתנו")
- פסקת סיכום קצרה ותחושה של "סגרנו את הנושא"

לא מסיימים עם סיכום מנופח.

---

## מבנה הפסקאות

**קצר מאוד.** כל מחשבה — שורה אחת.
לפעמים **מילה אחת.** (`"רגע."` — להשהייה דרמטית)
לפעמים **שאלה ותשובה.**

```
כן.
אבל...
אז מה עושים?
```

**אין בלוקים ארוכים של טקסט.** אם יש יותר מ-4 שורות רצוף — שוברים.

**בשורות ריקות** בין רעיונות — כן. זה לא בזבוז. זה מקצב.

## ניסוחים ואוצר מילים

### מועדף

| במקום | נכתוב |
|-------|--------|
| "לסיכום" | "בקיצור" / "בגדול" |
| "לאור האמור לעיל" | "לכן" / "אז" |
| "ניתן לראות כי" | "אפשר לראות ש" / "תראו" |
| "על מנת ל-" | "כדי ל-" |
| "הינו" / "הינם" | "הוא" / "הם" |
| "כפי שצוין" | "כמו שאמרנו" / "כבר אמרנו, לא?" |
| "מומלץ לבצע" | "כדאי לעשות" / "אני ממליץ" |

**מילים שאנחנו אוהבים:**
`בגדול`, `בקיצור`, `אגב`, `לכאורה`, `מסתבר ש`, `נו`, `יאללה`, `מגניב`, `כאב ראש`, `חלק`, `למעשה`

**מבנים שאנחנו אוהבים:**
- שאלה רטורית: "נשמע חלום, לא?" / "אבל מי רוצה להיחסם?"
- הכרה כנה: "לכאורה חלום ורוד, אבל מסתבר ש..."
- שבירת הרביעי: "כבר אמרנו, לא?" / "תרשמו לי בתגובות"
- הסתייגות עם הומור: "כך זה. מטא." / "ואם תשאלו..."

### אסור

- שפה פורמלית / משפטית / אקדמית
- "הינו", "הינם", "לאור", "על מנת", "בהתאם לכך"
- פסקאות של יותר מ-5-6 שורות ברצף
- פתיחה עם "במאמר זה נסקור..."
- סיום עם "לסיכום, ראינו כי..."
- מחמאות עצמיות ("מאמר מקיף", "הסבר מעמיק")
- שפת מכירות זולה ("הזדמנות פז!", "אל תפספסו!")

## כלים טכניים

### כותרות
`##` לחלקים גדולים.  
`###` לתתי-חלקים.  
לא יותר מ-2 רמות בדרך כלל.

### הדגשה
`**מילה**` — לדגשת מונח חשוב, לא לקישוט.  
`*הערה*` — להסתייגות/הבהרה.

### מקפים
`—` (em dash) לשבירת מחשבה: "כלי נהדר — אבל רק אם..."  
`-` (hyphen) ברשימות.

### סוגריים
לאסידות קצרות, הבהרות, או הומור עצמי:  
`(תשאלו את המזכירה מה הם הכי שונאים לעשות...)`

### שאלות רטוריות
פותחים בהן, ועונים. לא משאירים ריחוף:
```
אז מה עושים כשמקבלים קואורדינטות?
נעזרים ב-OpenStreetMap.
```

## מספרים, מחירים, נתונים טכניים

- מספרים — בספרות, לא באותיות: "24 שעות", "250 הודעות"
- מחירים — עם הקשר: "בין כמה עשרות דולרים לכמה מאות שקלים"
- שמות מוצרים / כלים — כפי שהם, בלי עיברות: Fillout, Priority, Engini, WhatsApp API

## בקצרה — מה עושה טקסט טוב בסגנון הזה

1. **פותח עם הכאב** — שאלה שהקורא שואל לעצמו
2. **מסביר בשכבות** — מהפשוט למורכב, בלי לדלג
3. **כן עם המגבלות** — לא מוכר, מסביר
4. **קצר** — כל שורה מרוויחה את מקומה
5. **מסיים קל** — לא הרצאה, שיחה
