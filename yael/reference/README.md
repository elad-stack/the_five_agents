# תיקיית דוגמאות — Reference

> הוסף לכאן קבצי `.md` עם דוגמאות לטקסטים בסגנון הבית שלנו.

יעל קוראת את כל הקבצים בתיקייה זו בתחילת כל משימת שכתוב,
ומשתמשת בהם כדי לדייק את הסגנון.

## מוסכמות

- שם הקובץ: תיאורי, למשל `example-blog-post.md`, `example-linkedin.md`
- כותרת בראש הקובץ המסבירה מאיזה פורמט/ערוץ הדוגמה
- ניתן להוסיף הערה קצרה למה הדוגמה הזו מייצגת את הסגנון שלנו
