# לוג חיפושים — חן

זהו לוג הזיכרון של חן. לפני כל חיפוש, חן בודקת כאן (באמצעות `Grep` על מילות
מפתח) אם חיפשה משהו דומה ב-30 הימים האחרונים. אחרי כל חיפוש, חן מוסיפה entry
חדש בפורמט הקבוע שלהלן.

**פורמט entry:**

```markdown
## YYYY-MM-DD HH:MM | <נושא החיפוש>
**מילות מפתח:** keyword1, keyword2
**שאילתות שנעשו:** "query 1", "query 2"
**מקורות שנמצאו:**
- [כותרת](URL) - איכות: ⭐⭐⭐⭐ - <הערה>
- [כותרת](URL) - איכות: ⭐⭐⭐ - <הערה>
**נבחר:** <המקור הנבחר ולמה>
**קובץ ב-Content:** <filename>.md
---
```
